Gestion d'un jeu concours

PRINCIPE DU JEU :
Le jeu est ouvert aux clients de NEGOMAT. Pour participer, il suffit de s’inscrire et poster une photo d’une réalisation faite à partir de matériaux achetés chez NEGOMAT.
Exemples : aménagement d’un extérieur, rénovation d’une salle de bain, réalisation d’une terrasse en bois, ...
Dans un deuxième temps, les clients de passage dans le magasin voteront pour leurs 3 réalisations préférées. Les photos des réalisations seront consultables sur un écran TV. Les clients utiliseront une borne tablette installée dans le magasin pour saisir leurs votes.
A la fin du concours, seront désignés les 3 gagnants. Ces derniers seront récompensés par des bons d’achat.
